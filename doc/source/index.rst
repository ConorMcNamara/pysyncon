pysyncon
========

pysyncon is a Python package that provides methods for the synthetic control
method due to `Abadie, Diamond & Hainmueller <http://dx.doi.org/10.1198/jasa.2009.ap08746>`_,
the augmented synthetic control method due to
`Ben-Michael, Feller & Rothstein <http://doi.org/10.3386/w28885>`_ and for
performing placebo tests.
